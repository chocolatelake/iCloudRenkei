# TToDo 起動手順書

TToDo（Discord Bot + Web Dashboard）を起動し、外部からアクセス可能にするための手順です。
**2つのターミナル（黒い画面）** を使用します。

## 1. アプリケーション本体の起動

**ターミナル1** で実行します。

1. プロジェクトのフォルダ（`TToDo` フォルダなど）へ移動します。
2. 以下のコマンドを実行し、ローカルサーバーを立ち上げます。

```powershell
dotnet run

```

* **確認:** `Now listening on: http://localhost:5000` と表示されれば成功です。
* **注意:** この画面は閉じずにそのままにしておきます。

---

## 2. 外部公開用トンネル (ngrok) の起動

**ターミナル2**（新しいウィンドウ）で実行します。

1. 以下のコマンドをコピー＆ペーストして実行します。
※URLの入力ミスを防ぐため、コピペ推奨です。

```powershell
ngrok http --domain=oxydasic-unransomable-mellie.ngrok-free.dev 5000

```

* **ポイント:** ドメイン指定に `https://` や末尾の `/` を含めないでください。
* **確認:** `Status: online` と表示されれば成功です。

---

## 3. Webダッシュボードへのアクセス

## トラブルシューティング

* **ERR_NGROK_3200 (The endpoint is offline)**
* 原因: アプリ本体（手順1）が動いていないか、ngrok（手順2）が止まっています。
* 対策: 両方のターミナルがエラーなく動いているか確認してください。


* **ERR_NGROK_314 / 9033 (Non-numeric port / Paid plan required)**
* 原因: コマンドのドメイン指定に `https://` や `/` が混ざっています。
* 対策: 手順2のコマンドを正確にコピペし直してください。


* **Botが反応しない**
* 原因: `dotnet run` の画面でエラーが出て止まっている可能性があります。
* 対策: ターミナル1を確認し、エラーが出ていれば再起動してください。
