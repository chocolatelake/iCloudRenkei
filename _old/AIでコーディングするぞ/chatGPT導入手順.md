GPT-5.2 統合環境 導入完全マニュアル
目的: 奥村研データで「最強」と証明された gpt-5.2 を、PCとスマホの両方で使い倒す。
予算: 月額 $20 (約3,000円) 以内。
構成:
 * 脳みそ: OpenAI API (gpt-5.2)
 * PC: VS Code + Cline (コーディング用)
 * スマホ: Chatbox AI (思考整理用)
Step 1: 脳みそ (OpenAI API) の契約と設定
まずは「GPT-5.2」を使うための権利（APIキー）を取得し、財布の設定をします。
 * OpenAI Platform にアクセス
   * ブラウザで platform.openai.com を開きます。
   * 右上の「Sign up」からアカウントを作成（または「Log in」）します。
 * クレジットカードの登録
   * 左側のメニューバーにある歯車アイコン 「Settings」 をクリックします。
   * 「Billing」 タブを選択します。
   * 「Add payment details」 をクリックし、クレジットカード情報を入力して登録します。
 * プリペイドチャージ (重要)
   * 「Add to credit balance」 というボタンをクリックします。
   * 金額入力欄に 10 (ドル) と入力します。
   * 支払いを完了させます。これで $10 分のクレジットがチャージされました。
   * ※後払いではなく先払いにすることで、使いすぎを物理的に防ぎます。
 * 安全装置 (Usage Limits) の設定
   * 左メニューの 「Settings」 > 「Limits」 を選択します。
   * 「Monthly budget (Usage limit)」 の欄に 20 と入力して保存します。
   * これで、万が一プログラムが暴走しても、$20（約3,000円）に達した瞬間に停止するため、高額請求のリスクがゼロになります。
 * APIキーの発行
   * 左上の 「Dashboard」 をクリックしてトップに戻ります。
   * 左メニューの 「API keys」 をクリックします。
   * 右上の 「+ Create new secret key」 をクリックします。
   * Name: わかりやすく Mac_iPhone_Dev などと入力します。
   * Create secret key をクリックすると、sk- から始まる長い文字列が表示されます。
   * 【最重要】 このキーをコピーして、メモ帳などに一時的に貼り付けておいてください。（この画面を閉じると二度と表示されません！）
Step 2: 母艦 (Mac mini) の環境構築
Mac mini に VS Code と Cline を導入し、コーディング環境を作ります。
 * VS Code のインストール (まだの場合)
   * 公式サイト code.visualstudio.com から「Mac Universal」版をダウンロードしてインストールします。
 * 拡張機能「Cline」のインストール
   * VS Code を起動します。
   * 左端のアイコンバーから 「拡張機能 (四角いブロックのアイコン)」 をクリックします（ショートカット: Cmd + Shift + X）。
   * 検索バーに Cline と入力します。
   * リストに出てきた 「Cline」 (通常は一番上) の 「インストール」 ボタンをクリックします。
 * Cline と GPT-5.2 の接続設定
   * インストールが完了すると、左端のアイコンバーにロボットのようなアイコン（Cline）が追加されるのでクリックします。
   * Clineのパネルが開いたら、上部にある 歯車アイコン (Settings) をクリックします。
   * 以下の通りに設定を入力します。
     * API Provider: ドロップダウンから OpenAI を選択。
     * API Key: Step 1 でコピーした sk- から始まるキーを貼り付け。
     * Model ID: 入力欄に手動で gpt-5.2 と入力。
       * ※ gpt-5.2-pro と間違えないよう注意！
 * カスタム指示 (Custom Instructions) の入力
   * 同じ設定画面の下の方にある 「Custom Instructions」 欄に、以下のテキストをコピーして貼り付けます。
     * ※これにより、AIの回答品質とコーディングの正確性を高めます。

Step 3: 外出用端末 (iPhone 16 Pro) の環境構築
iPhone で思考整理や壁打ちができるようにします。
 * アプリのインストール
   * App Store を開き、検索タブで 「Chatbox AI」 を検索します。
   * 「Chatbox AI - Chat with LLM」 (または類似の名称) を入手（インストール）します。
 * APIの設定
   * アプリを起動します。
   * 左上のメニューボタン（または設定アイコン）をタップし、「設定 (Settings)」 を開きます。
   * 「AIモデル (AI Model)」 または 「モデル設定」 をタップします。
   * 「AIプロバイダー (AI Provider)」 で OpenAI API を選択します。
   * 「APIキー (API Key)」 の欄に、Step 1 で発行した sk- から始まるキーを貼り付けます。
     * ヒント: Mac mini でキーをコピーしておけば、Apple製品の連携機能（ユニバーサルクリップボード）により、iPhone側でそのまま「ペースト」できます。
 * モデルの選択
   * 「モデル (Model)」 のリストを開きます。
   * リストの中に gpt-5.2 があれば選択します。
   * もしリストにない場合は、リストの一番下などにある「カスタムモデルを入力」のような欄に、手動で gpt-5.2 と入力して保存します。
 * 動作確認
   * チャット画面に戻り、「こんにちは、モデル名を教えて」と送信してみます。
   * 「私はGPT-5.2です」等の返答があれば成功です。
完了：運用開始
これで、以下の環境が整いました。
 * Mac mini: VS Code + Cline で、ローカルファイルを直接ガリガリ編集できる開発環境。脳みそは最新の gpt-5.2。
 * iPhone 16 Pro: いつでもどこでも gpt-5.2 の400kコンテキスト（超記憶力）を使って、プロジェクトの設計やアイデア出しができる環境。
 * お財布: 月額上限 $20 でガッチリ守られており、使いすぎの心配なし。
さあ、開発を始めましょう！
