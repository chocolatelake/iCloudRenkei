# カフェ開発用 Mini PC 構成リスト

## 1. PC本体（主役）
* **商品名検索ワード**: `GMKtec NucBox G3 N100 16GB 512GB`
* **選ぶべきスペック**:
    * CPU: Intel N100
    * メモリ: 16GB（必須）
    * SSD: 512GB
    * OS: Windows 11 Pro 搭載
* **予算感**: 22,000円 〜 26,000円（クーポン適用時）
* **備考**: 「NucBox G3」という青緑または黒のモデルが一番コスパが良いです。

## 2. iPad モニター化セット（映像）
* **商品名検索ワード**: `HDMI キャプチャ USB Type-C 1080p 60fps`
* **選ぶもの**: 2,000円〜3,000円くらいの安い製品。iPadの充電端子（USB-C）に挿さるもの。
* **予算感**: 約2,000円

* **商品名検索ワード**: `HDMIケーブル 0.5m`
* **選ぶもの**: PCとキャプチャボードを繋ぐ用。長いと邪魔なので「0.5m」か「1m」の短いものがおすすめ。
* **予算感**: 約800円

## 3. モバイル電源セット（重要）
※ カフェのコンセントを使わず、モバイルバッテリーで動かすための魔法のセットです。

* **商品名検索ワード**: `USB PD トリガーケーブル 12V 5.5 2.5`
* **選ぶもの**: 片方がUSB-C、片方が丸いプラグ（DCプラグ）のケーブル。
    * **重要**: 必ず「12V」と書いてあるもの。
    * **サイズ**: GMKtec G3は「外径5.5mm / 内径2.5mm」が一般的です。
* **予算感**: 約1,000円

* **商品名検索ワード**: `CIO モバイルバッテリー 65W` （または Anker 65W）
* **選ぶもの**: 必ず「PD 65W」出力に対応しているもの。
    * ※もし既にお持ちのバッテリーが「PD対応 45W以上」なら、それを試してから買ってもOKです。
* **予算感**: 約5,000円〜7,000円（持っていなければ）



# 届いたらやることリスト

## STEP 1: 動作確認
1.  家のテレビやモニターに、付属のACアダプタでPCを繋いで電源ON。
2.  Windows 11の初期設定（言語設定、Wi-Fi接続など）を済ませる。
3.  **重要**: アップデートを一通り終わらせておく。

## STEP 2: iPad開発環境の構築
1.  iPadにアプリ「**Camo Studio**」または「**Genki Studio**」をインストール（無料）。
2.  以下の順で接続する。
    `[PC] -> [HDMIケーブル] -> [キャプチャボード] -> [iPad (USB-C)]`
3.  電源をモバイルバッテリー構成に切り替える。
    `[バッテリー] -> [トリガーケーブル] -> [PC (電源ポート)]`
4.  PCのスイッチON。iPadのアプリを開くと、Windowsの画面が映ります。

## STEP 3: Linux開発環境 (WSL2) の導入
Windows上で黒い画面（ターミナル）を使えるようにします。

1.  PC画面左下の検索バーで「PowerShell」と検索し、「管理者として実行」する。
2.  以下のコマンドをコピー＆ペーストしてEnterキーを押す。
    `wsl --install`
3.  PCを再起動する。
4.  再起動後、自動的に「Ubuntu」のインストールが始まるので、好きなユーザー名とパスワードを決める。

👉 **これで「Windowsの中に完全なLinux環境」が完成です！**
