# 北斗の作成物

## 1. Switch遠隔操作・自動化システム
外出先やPCからNintendo Switchを操作・自動化するためのハードウェア/ソフトウェア統合環境。
- **概要**: スマホ → PC (Tailscale/Sunshine) → 自作C#アプリ → Raspberry Pi Pico → Switch という経路で制御信号を送る。
- **構成要素**:
    - **ハードウェア**: Raspberry Pi Pico (Arduino HID化), USBシリアル変換モジュール, Windows PC (Host)
    - **ソフトウェア (制御)**:
        - **Firmware**: Arduino IDEで作成した `.ino` (Switchコントローラーエミュレータ)
        - **Host App**: Visual Studio (C#) で作成したGUIアプリ。シリアル通信でPicoに指令を送る。
    - **リモート環境**:
        - **Tailscale**: 外部から安全に自宅LANへアクセスするメッシュVPN。
        - **Sunshine / Moonlight**: 低遅延・高画質なゲームストリーミング配信。
        - **Wake on LAN / SwitchBot**: 物理的な電源管理による冗長化。

## 2. TToDo
DiscordとWebを連携させた、ハイブリッドなタスク管理・日報自動化ボット。
- **概要**: チャットでの気軽な入力と、Webカンバンボードでの視覚的な整理を両立。
- **主な機能**:
    - **Discord Bot**: `!ttodo` コマンドによるタスク登録、スマート解析（優先度・タグ自動判定）。
    - **Web Dashboard**: ドラッグ＆ドロップ対応のカンバンボード（Active / Done / Archive）。
    - **自動化**: 完了タスクを集計し、毎日決まった時間に日報を自動生成・投稿。
- **技術スタック**:
    - **言語**: C# (.NET / Discord.Net)
    - **データ**: JSON (System.Text.Json)
    - **フロントエンド**: Web (Responsive Design)
